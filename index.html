Bluebike Usage and Weather Analysis

1. Correlation Heatmap: Weather and Usage

This heatmap illustrates the correlation between Bluebike usage metrics and various weather conditions. Strong positive correlation exists between member ride counts and total travel time. Weekend rides tend to be longer and more common among casual riders.

2. Interactive Line Chart: Hourly Usage Trends by Weather

This interactive line chart shows average Bluebike ride counts throughout the day under different weather conditions. Clear and Fair weather see sharp morning and evening peaks, while Rain and Snow significantly reduce usage.

3. Interactive Scatter Plot: Temperature vs Ride Count

This interactive scatter plot shows the relationship between temperature and Bluebike ride volume, categorized by weather condition. You can use the radio button to filter by specific weather types. This view highlights how warmer days generally encourage more rides, especially under Clear or Fair skies.

4. Rider Comparison by Weekday: Member vs Casual

This interactive bar chart compares member and casual Bluebike users across days of the week. Select a day from the dropdown to explore how ride counts and ride durations vary by user type. Casual riders tend to have longer ride durations, while members show higher volume on weekdays.

5. Bluebike Ride Count by Neighborhood

This interactive map shows the Bluebike ride counts across Boston's neighborhoods. The color intensity reflects the number of rides, with darker areas indicating higher ride volumes. You can zoom in and click on each neighborhood for more details.
